This starter project will allow you to see a few common Ionic patterns.

If you'd notice, the source for this page has an ion-header with an ion-navbar; that informs Ionic that if this is not the root page of the application to show a back button to navigate through the history backwards and go back to the previous component rendererd.

You may also notice the menuToggle attribute on the button in the ion-header; that will allow us to toggle the sidemenu, which is rendered in the app component.
